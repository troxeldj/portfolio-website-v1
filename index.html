Projects I'm proud of

Assault Cube Trainer

Description: This is a trainer (or cheat) for an FPS game called assault cube. In this project, game memory addresses and offsets were found using cheat engine and x86dbg. After finding these offsets, win32 API functions were called to obtain a process ID, process handle, and read and write to the processes space in virtual memory. With this, we are able to do anything we want. Unlimited health, unlmited armor; we can even call in-game functions!

Technologies Used:

  • Cheat Engine
  • x86dbg
  • C
  • C++
  • x86 Assembly
